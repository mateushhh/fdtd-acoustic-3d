# Symulator propagacji fal akustycznych metodą FDTD w przestrzeni 3D

### Podsumowanie
Zadaniem projektowym jest opracowanie aplikacji wizualizującej proces rozchodzenia się fal akustycznych oraz ich interackcji z przeszkodami w przestrzeni 3D
